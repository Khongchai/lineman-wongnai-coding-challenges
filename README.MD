# How to run everything

### api-gateway (flask)

```shell

pip install requirements.txt

cd flaskr

flask_app=main

flask_debug=1 flask run

```

## json server

```shell

cd json-server

npm install

npm start

```

## react

```shell

cd web

npm install

npm start

```

## Other info

create-react-app addons:

- Chakra UI
- TypeScript
